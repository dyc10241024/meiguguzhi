# 需求方案

## 背景

支付宝、天天基金、银行渠道里买到的大多数 QDII 基金属于场外基金。场外基金没有盘中实时成交价，投资者最终按基金公司披露的每日净值结算。

由于监管和平台策略变化，很多平台已经弱化或下架基金实时估值。现有估值工具一般也是通过公开持仓、指数行情、汇率和历史误差进行算法估算。

本项目目标是做一个个人自用工具：不发布小程序，不依赖后端数据库，直接在 Cursor / VSCode 中刷新查看指定 QDII 基金估值。

## 产品定位

名称暂定：美股 QDII 估值助手。

运行形态：Cursor / VSCode 扩展，本地安装，自用。

用户：主要是项目作者本人。

核心价值：

- 快速查看纳指、纳指 100、标普 500、美元人民币汇率等参考指标。
- 查看指定 QDII 基金的估算涨跌幅。
- 点击基金后查看前十大持仓、权重、个股涨跌幅、贡献值。
- 对比历史实际净值涨跌幅，观察算法误差。

## 核心页面

扩展提供一个侧边栏 Webview。

顶部指标区：

- 纳斯达克指数
- 纳斯达克 100 指数
- 标普 500 指数
- 美元人民币汇率
- 数据更新时间

基金列表区：

- 基金名称
- 基金代码
- 估算涨跌幅
- 最近实际净值日期
- 最近实际涨跌幅
- 估算误差提示

基金详情弹窗或详情面板：

- 前十大持仓
- 股票代码
- 股票名称
- 市场
- 持仓权重
- 当日涨跌幅
- 加权贡献
- 数据源状态

历史回测区：

- 最近 20 个净值日
- 实际涨跌幅
- 原始估算涨跌幅
- 调优后估算涨跌幅
- 单日误差
- MAE、RMSE、方向正确率

## 用户流程

1. 打开 Cursor / VSCode。
2. 点击侧边栏中的“美股估值”图标。
3. 扩展自动加载默认基金列表。
4. 用户点击刷新。
5. 扩展实时请求行情、持仓、历史净值并计算估值。
6. 用户点击某只基金，查看持仓贡献和历史误差。

## 非目标

当前阶段不做：

- 用户登录。
- 云端数据库。
- 小程序审核和发布。
- 自动交易或投资建议。
- 预测最终基金净值。
- 多账户持仓管理。

## 风险提示

页面必须明确标注：

“场外基金没有真正实时行情，当前结果为基于公开持仓和市场行情的算法估算，不代表基金公司最终披露净值。”
