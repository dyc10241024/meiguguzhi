# 美股 QDII 估值助手

个人自用的 Cursor / VSCode 扩展，用于查看美股 QDII 场外基金的估算涨跌幅、持仓贡献与历史回测误差。

场外 QDII 基金没有真正的盘中实时净值，本工具展示的是**算法估算**：用公开持仓、股票收盘涨跌、美元人民币汇率与历史净值误差拟合，估算下一次净值披露前可能接近的涨跌幅。不代表基金公司最终披露净值。

完整规格见 [`docs/SPEC.md`](docs/SPEC.md)。

## 界面

自上而下三区：顶部指标区 → 算法 Tab → 基金列表。

- **顶部指标区**：纳斯达克指数、纳斯达克100指数、标普500指数、美元人民币汇率、数据更新时间、刷新按钮。
- **算法 Tab**：算法1（全量持仓估值）、算法3（快速估值）。
- **基金列表**（仅核心结果）：基金名称、基金代码、最新估值涨跌幅、估值时间、数据获取时间。
- **基金详情弹窗**：算法3 展示前十大持仓；算法1 展示最新报告的全部持仓。每行含名称、代码、市场、占净值比例、当日涨跌幅、贡献值、数据来源、数据获取时间。
- **历史回测弹窗**：头部展示 alpha、beta、MAE、RMSE、方向正确率、数据获取时间；正文展示每个净值日的实际涨跌、估值涨跌与每日误差。

关注基金：广发全球精选 `270023`、长城全球新能源车 `501226`、华宝纳斯达克精选 `017436`。

## 算法

### 算法3：快速估值

持仓取最新季度前十大，按前十大权重归一化后代理整个股票类仓位，叠加汇率贡献，再用历史回测拟合的 alpha/beta 校准。

```
raw估值  = (Σ(w_i × r_i) / 前十大权重合计) × 股票类资产占比 + 汇率贡献
最终估值 = alpha + beta × raw估值
```

### 算法1：全量持仓估值

持仓采用混合：刷新时从当前日期往回寻找最近可获取的全量披露底仓，再拉最新十大，用最新十大按 secid 替换全量里的同股票，按权重重排后计算。与算法3 同构，仅数据覆盖更大。

```
raw估值  = (Σ(w_i × r_i) / 全量持仓权重合计) × 股票类资产占比 + 汇率贡献
最终估值 = alpha + beta × raw估值
```

### alpha / beta 与误差指标

alpha、beta 由历史回测自动拟合（`beta = cov(raw, actual)/var(raw)`，`alpha = mean(actual) − beta × mean(raw)`）。MAE、RMSE、Bias、方向正确率仅用于评估误差，不参与估值公式。

## 开发与调试

```bash
npm install
npm run compile
```

在 Cursor / VSCode 中：

1. 打开 `D:\meiguguzhi`。
2. 执行 `npm run compile`。
3. 按 `F5` 启动扩展开发宿主。
4. 在新窗口打开「美股估值 / QDII 估值」界面。

## 目录

- `src/extension.ts`：激活、注册 Webview 与命令。
- `src/webview/QdiiViewProvider.ts`：Webview 承载与取数推送。
- `scripts/live-data.js`：取数与估值计算（子进程）。
- `media/main.js`、`media/main.css`：前端渲染。
- `data/funds.json`：基金列表与资产配置参数；历史净值和历史估值序列刷新时从接口动态获取并计算。
- `docs/SPEC.md`：正式第一版规格文档。
